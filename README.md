# Oozing Floor

It is a minigame with a theme *Everything is Changing*. 

## Features:

### 1. Flipping Surfac
Every 30 seconds, the entire arena floor swaps between *Slime* and *Honey* blocks.
- Slime has high bounce and one knockback hit can send you across the map.
- Honey has sticky movement and reduced jump height!

### 2. Spawning Slime
Elimination isn't the ending factor, evertime a player falls into the void, a Size 2 Slime spawns at the center of the map which makes the map more "living".

### 3. Oozing Effect
Every 20 seconds, the server gives Oozing Effect to all players, and taking damage during this phase will cause more slimes to spawn.


## Combat Kit
**Knockback II Stick**
**Infinite wind charges**
